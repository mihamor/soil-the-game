# Soil The Game
Simple sandbox-adventure game written in C++/SFML

### Controls
W,A,S,D - movement
I - inventory 
C - craft menu
Right click - place block
Left click - destroy block/use item

### Saves
There are 3 slots available to save your in-game progress
